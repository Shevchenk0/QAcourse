1) Посмотреть где я - pwd
2) Создать папку - mkdir foldermane
3) Зайти в папку - cd foldermane
4) Создать 3 папки - mkdir foldermane1 foldermane2 foldermane3
5) Зайти в любую папку - cd foldermane2
6) Создать 5 файлов (3 txt, 2 json) -touch 1.txt 2.txt 3.txt 4.json 5.json
7) Создать 3 папки - mkdir foldermane4 foldermane5 foldermane6
8. Вывести список содержимого папки - ls -la
9) + Открыть любой txt файл - vim 1.txt
10) + написать туда что-нибудь, любой текст - режим insert, пишем необходимое
11) + сохранить и выйти - режим Esc значение :wq
12) Выйти из папки на уровень выше - cd ..
13) переместить любые 2 файла, которые вы создали, в любую другую папку - mv 1.txt 2.txt foldermane4
14) скопировать любые 2 файла, которые вы создали, в любую другую папку cp 3.txt foldermane5
15) Найти файл по имени - find .name 31.txt
16) просмотреть содержимое в реальном времени (команда grep - осущ-ет поиск текста по файлам) изучите как она работает - 
ключи -v(ищет строки не соответствующие шаблону, т.е. строки без слова которого ищем)/-n(выводить построчно) (т.е. grep -v/grep -n)
https://aidalinux.ru/w/Grep 
17) вывести несколько первых строк из текстового файла - grep -n Hello 31.txt
18) вывести несколько последних строк из текстового файла grep -n you 31.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает - развитая команда для пролистывания текста.
Позволяет перематывать текст не только вперёд, но и назад, осуществлять поиск в обоих направлениях, переходить сразу в конец или в начало файла.
Команда не считывает текст полностью, а загружает его небольшими фрагментами
Выйти из "less" - q
20) вывести дату и время - date
